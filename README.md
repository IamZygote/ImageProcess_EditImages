# ImageProcess to Edit Images
Applying many diffrent Image processing filters on images

Made using **Matlab**

## Algorithms

- Median Filter
- Gaussian Filter
- Histogram
- Uniform Noise
- Interpolation (resize)
- Laplacian (Edge Sharpining)
- Sobel Edge (Edge Sharpining)
